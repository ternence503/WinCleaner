【如何使用 / How to Use】

  最簡單方式（推薦）：
  Easiest method (Recommended):

  1. 雙擊「啟動清理工具.bat」
     Double-click "啟動清理工具.bat"

  2. 出現 UAC 視窗時選「是 / Yes」（需要管理員權限）
     Click "Yes" when UAC prompt appears (admin required)

  3. 選 [1] 一鍵全部清理，等待完成即可
     Choose [1] Full Clean and wait for completion

  備用方式 / Alternative:
  右鍵點擊「WinCleaner.ps1」→「以 PowerShell 執行」
  Right-click "WinCleaner.ps1" → "Run with PowerShell"


【清理項目說明 / What Gets Cleaned】

  1.  使用者暫存檔         - %TEMP% 資料夾內的暫存檔
  2.  系統暫存檔           - Windows\Temp 資料夾
  3.  Windows 更新快取     - 已下載的更新安裝包（可重新下載）
  4.  資源回收筒           - 清空所有磁碟的資源回收筒
  5.  瀏覽器快取           - Chrome / Edge / Firefox / IE / Brave
  6.  App 快取             - LINE / Discord / Teams / Slack / Zoom / Steam
  7.  Delivery Optimization - Windows P2P 更新快取
  8.  預先擷取檔案         - Windows\Prefetch（會自動重建）
  9.  縮圖快取             - 檔案總管縮圖（會自動重建）
  10. DNS 快取             - 網路 DNS 記錄（有助解決網路問題）
  11. 字型快取             - 字型資料庫（會自動重建）
  12. 錯誤報告             - Windows 錯誤記錄檔
  13. 舊版 Windows         - Windows.old 資料夾（需手動確認）
  14. 系統日誌             - 系統事件記錄檔
  15. 記憶體最佳化         - 釋放閒置的 RAM


【支援系統 / Supported Systems】

  ✓ Windows 7
  ✓ Windows 8 / 8.1
  ✓ Windows 10（所有版本）
  ✓ Windows 11（所有版本）


【安全說明 / Safety Notes】

  ✓ 不會刪除您的個人檔案（文件、圖片、影片等）
  ✓ 不會刪除 LINE 對話紀錄、Discord 設定、瀏覽器書籤密碼
  ✓ 所有刪除操作都有錯誤處理，不會因為檔案被占用而崩潰
  ✓ Windows.old 刪除需要您手動確認
  ✓ 啟動項目停用可隨時從工作管理員還原
  ✓ 清理完成後建議重新啟動電腦

  It will NOT delete your personal files (documents, photos, videos)
  It will NOT delete LINE chats, Discord settings, or browser bookmarks
  All deletions handle errors gracefully
  Windows.old deletion requires your confirmation
  Restart recommended after cleaning


【常見問題 / FAQ】

  Q: 執行後顯示「此系統上禁止執行腳本」
  A: 使用「啟動清理工具.bat」而非直接執行 .ps1

  Q: Script cannot be run on this system
  A: Use "啟動清理工具.bat" instead of running .ps1 directly

  Q: 清理後重開機桌面花一點時間載入？
  A: 正常！縮圖和圖示快取正在重建，只需一次。

  Q: Desktop takes longer to load after restart?
  A: Normal! Thumbnail and icon cache is rebuilding. One-time only.

  Q: 可以定期使用嗎？
  A: 可以！建議每 1~2 個月清理一次效果最好。

  Q: Can I use this regularly?
  A: Yes! Recommended once every 1-2 months for best results.
